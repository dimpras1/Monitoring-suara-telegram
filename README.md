# Monitoring-suara-telegram
Proyek ini adalah sistem monitoring kebisingan ruangan kelas berbasis ESP32 yang menggunakan sensor mikrofon (GY-MAX4466 / MAX9814) untuk membaca tingkat suara secara real-time. Ketika kebisingan melebihi batas tertentu, sistem akan mengirimkan notifikasi otomatis ke Telegram sebagai peringatan.
